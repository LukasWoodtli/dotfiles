
# Install

```sh
git clone https://github.com/LukasWoodtli/dotfiles.git ~/.dotfiles && \
cd ~/.dotfiles && \
git submodule update --init --recursive && \
./install
```


```sh
# update
cd ~/.dotfiles && \
git pull && \
git submodule update --init --recursive && \
./install
```
